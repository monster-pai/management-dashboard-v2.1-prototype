# 管理驾驶舱 V2.1 - PC端前端原型

> 基于三层文档架构（底表→视图→门户）的PC端管理驾驶舱前端原型。
> 使用真实数据（fund-data.js）驱动，完整复刻移动端v5.html的交互与视觉。

## 仓库结构

```
├── index.html              # 主页面（门户层）
├── css/
│   └── styles.css          # 全局样式（复刻v5.html卡片风格）
├── js/
│   ├── data.js             # 真实数据（fund-data.js）
│   ├── app.js              # 主应用逻辑（路由、状态、初始化）
│   ├── charts.js           # 图表渲染引擎（Chart.js封装）
│   ├── utils.js            # 工具函数（格式化、计算）
│   └── modules/
│       ├── flash-fund.js   # 快报-资金模块（V-D-001~V-D-005）
│       ├── monthly-finance.js  # 月报-财务模块（V-01-001~V-03-003）
│       ├── monthly-trading.js  # 月报-成交模块（V-04-001~V-05-002）
│       └── drill-tables.js     # 下钻辅助视图（D-01-001~D-04-001）
├── docs/
│   └── 三层文档映射.md      # 视图编号与代码文件的对应关系
└── README.md
```

## 三层编号体系

| 层级 | 编号格式 | 示例 | 说明 |
|------|---------|------|------|
| 第一层 | T-NN | T-01 | 底表（数据字段定义） |
| 第二层 | V-XX-NN | V-01-001 | 标准分析视图 |
| 第二层 | D-XX-NN | D-01-001 | 下钻辅助视图 |
| 第三层 | P-XX-NN | P-M-01 | 门户页面 |
| 第三层 | L-XX-NN | L-01-001 | 页面布局卡片 |

## 已实现视图

### 快报-资金（5个视图）
- V-D-001 监管指标KPI
- V-D-002 资金投向分布（饼图+表格，支持分类/项目切换）
- V-D-003 资金来源分布（饼图+表格，公司筛选联动）
- V-D-004 授信额度使用（进度条表+预警）
- V-D-005 未来提还款计划（甘特图）

### 月报-财务（12个视图）
- V-01-001 营业净收入KPI（可下钻公司明细）
- V-01-002 净利润KPI（可下钻公司明细）
- V-01-003 营业净收入趋势（双轴组合图）
- V-01-004 净利润趋势（双轴组合图，与V-01-003互斥）
- V-02-001 营业净收入构成表（6类，点击联动）
- V-02-002 收入饼图（双环形，扇区点击联动）
- V-02-003 分类营业收入趋势（双轴组合图）
- V-03-001 业管费支出构成表（2类，点击联动）
- V-03-002 费用饼图（双环形，扇区点击联动）
- V-03-003 业管费趋势（双轴组合图）
- D-01-001 公司明细钻取（7列表格，支持公司筛选→趋势图联动）
- D-02-001 收入项目明细钻取（视图模式切换：按项目/按公司）
- D-03-001 费用项目明细钻取（视图模式切换：按项目/按公司）

### 月报-成交（6个视图）
- V-04-001 成交KPI（3指标切换）
- V-04-002 成交量趋势（双轴组合图）
- V-04-003 市占率趋势（单折线图）
- V-05-001 分交易所明细（6家，点击联动）
- V-05-002 分交易所饼图（双环形，扇区点击联动）
- D-04-001 品种明细钻取（更多/收起）

## 交互特性

- 模块切换：快报/月报/年报/考核/对标
- 子Tab切换：财务/成交/手续费/权益
- 期间导航：月份/年度切换
- 公司筛选：全部公司/期货/投资/国贸/资本
- 四层联动：分类行→饼图→趋势图→项目钻取
- Toggle行为：再次点击取消选中状态

## 数据来源

使用 `fund-data.js`（从看板数据样例_样本数据.xlsx自动生成）作为真实数据驱动。

## 技术栈

- HTML5 + CSS3 + Vanilla JavaScript（无框架依赖）
- Chart.js 4.4.7（图表库）
- 模块化ES5（无构建工具，浏览器直接运行）
